designer_ext is the extended version of fpGUI uiDesigner.

It needs the last fpGUI version =>
https://github.com/graemeg/fpgui/

With run-only-once, window list, undo feature, integration into IDE, editor launcher,...

How to install it ?

Just compile /designer_ext/src/designer_ext.pas

If you want assistive for designer_ext => copy /designer_ext/bin/your_system/sak folder => into your main application folder.
Then in "Settings" => enable "Assistive".

For integation with fpc, Lazarus, Typhon, Geany, external-code-editor, ..., you may use the patches...
Those patches are in /designer_ext/patch.

Please read the readme-patch-files before to apply patch.
